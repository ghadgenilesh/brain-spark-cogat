🌐 Free hosting options
🥇 Recommended: Cloudflare Pages (free forever)
  1. Create account at pages.cloudflare.com
  2. Click Create a project → Upload assets
  3. Upload cogat-brainspark.html and questions.json into the same folder
  4. Set your project name → Deploy
  5. Update DATA_SOURCE in the HTML to ./questions.json
  6. When you have new questions: re-upload only questions.json — app auto-refreshes on next open
🥈 Netlify Drop (free, no account needed to try)
  1. Go to app.netlify.com/drop
  2. Drag a folder containing both files onto the page
  3. Get a live URL instantly — no signup required
  4. Create a free account to keep the URL permanent and redeploy
🥉 GitHub Pages (free, good for version control)
  1. Create a free GitHub account and new public repo
  2. Upload both files to the repo root
  3. Go to Settings → Pages → Source: main branch
  4. Your app lives at username.github.io/repo-name
  5. Update questions: edit questions.json in the repo — app refreshes automatically
☁️ AWS S3 + CloudFront (original plan, still free tier)
  1. Create S3 bucket → enable static website hosting → public read
  2. Upload both files. Set DATA_SOURCE to your S3 URL
  3. Add CORS rule: allow GET from *
  4. Optional: add CloudFront CDN in front for global speed
  5. Free tier: 5 GB storage + 20K requests/month
